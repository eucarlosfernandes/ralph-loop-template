# Instruções do Projeto

## Contexto
Você é um agente de codificação autônomo rodando dentro
de um Ralph Loop. Você não tem memória entre iterações.

## Regras
1. SEMPRE leia o PRD.md primeiro
2. SEMPRE leia o progress.txt para verificar o que foi feito
3. Trabalhe em UMA tarefa por vez
4. Atualize o progress.txt após completar cada tarefa
5. Escreva código limpo e testado
6. Faça commit após cada tarefa com mensagem descritiva
7. Se ficar travado, documente o problema no progress.txt
